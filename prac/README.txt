Para copilar el programa:
     make

Para ejecutar el programa:
     ./prac

A continuacion el programa te preguntara que le digas una conjunto de ficheros para cargar (carpeta).
Seguidamente el programa te preguntara que valor quieres por k.
E igual por los valor t y por el numero de tiras.

Los conjuntos de ficheros utilizados son los siguientes:
-> DatosPrueba: utilizado para tester el algoritmo (caso de juguete)
-> DatosSimilares: utilizado para los analisis de precision
-> 50p5: conjunto de fichero con un 5% de palabras permutada
-> 50p25: conjunto de fichero con un 25% de palabras permutada
-> 50p50: conjunto de fichero con un 50% de palabras permutada
-> 50p75: conjunto de fichero con un 75% de palabras permutada
